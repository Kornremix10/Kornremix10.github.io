# A2 Personal Website

This repository contains my Assignment 2 personal website for *IAT 355*.  
The website presents a brief overview of who I am, my background, and a small visualization portfolio created using JavaScript and SVG.

---

## Website Structure

The project includes the following files:

- `index.html`  
  The main homepage of the website. It includes a short personal introduction, education, skills, experience, images of my work, and contact information.

- `visualizations.html`  
  A separate page dedicated to visualizations. This page contains:
  - One SVG data visualization
  - One creative SVG artwork  
  All visualizations are generated using JavaScript.

- `style.css`  
  Handles all styling for the website, including layout, colors, typography, and responsive design.

- `main.js`  
  Adds basic JavaScript interactivity to the homepage, including theme toggling and simple form validation.

- `vis.js`  
  Contains all JavaScript code used to generate SVG visualizations and creative SVG art.

- `images/`  
  Contains image assets used on the homepage to represent my work.
